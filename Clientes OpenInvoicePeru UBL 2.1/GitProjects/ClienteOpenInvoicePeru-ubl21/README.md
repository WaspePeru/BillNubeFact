# Cliente OpenInvoicePeru UBL 2.1
![Open Invoice Perú](https://raw.githubusercontent.com/erickorlando/facturacionelectronicasunat/master/openinvoiceperulogo.PNG "Open Invoice Perú")

Este proyecto, que se deriva del proyecto original [OpenInvoicePeru](https://goo.gl/9tuEiM). Ejemplifica como crear un cliente de la API REST tanto en C# y Visual Basic teniendo como base el .NET Framework 4.5.

## Paquetes Nuget ##

Puedes descargar los paquetes Nuget de **OpenInvoicePeru** desde la página oficial de [Nuget](https://www.nuget.org/packages/OpenInvoicePeru.UBL21/).
